# Projeto Cybersecurity DIO

## Meu Projeto de Força Bruta

### O que aprendi:
- Ataques de força bruta testam segurança de sistemas
- Ferramentas como Medusa automatizam testes de senha
- É importante ter ambientes seguros para testes

### Como se proteger:
- Usar senhas fortes e únicas
- Implementar bloqueio após tentativas erradas
- Usar verificação em duas etapas
- Monitorar logs de acesso

### Comando de exemplo:
medusa -h 192.168.1.100 -U usirariostxt -P senhas.txt -M ftp

**Projeto educativo para a formação de Cybersecurity da DIO**
